# Rental Dress — Dress & Suit Rental Platform

A full-featured web platform for a dress and suit rental business in Uruguay.
Includes a photo gallery and a multi-step booking system.

## 🔗 Live Demo
[rentaldress.vercel.app](https://rentaldress.vercel.app)

## 🛠️ Built With
- Next.js 14
- TypeScript
- Tailwind CSS

## ✨ Features
- Photo gallery showcasing dresses, suits and accessories
- Multi-step reservation system (category → date & time → contact info)
- Category selection: gowns, suits, accessories
- Appointment slots with available fitting rooms
- WhatsApp integration for accessories inquiries
- Minimal, editorial design optimized for mobile

## 📁 Project Context
Developed as a freelance project for a fashion rental business in Montevideo, Uruguay.
